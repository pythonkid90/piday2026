Nautilus Games

Nautilus Games is a small collaborative project/"company" by two children who made Scratch games during their 5th and 6th grade years. This site hosts a few of those projects for anybody to access. Nautilus Games is now defunct and this site will likely not be updated with new games until at least 2028.

Games

Fruit Clicker (v1.0.3)

Nautilus Technology's first published game (although most of the other games were created earlier). A simple clicker game (made in Scratch, technically Turbowarp). It is well polished and arguably Nautilus's best project.

Fruit Clicker (v0.9.1)

An earlier version of Fruit Clicker. It is less polished and has completely different code (written by a different Nautilus member), but it does have achievements, unlike v1.0.3. It is written in pure Scratch without any Turbowarp features.

1 BILLION Digits of Pi!!! (v1.0.0)

A sloppily put together Scratch program that shows the first billion digits of pi. Commentary: I realized that with the tools that TurboWarp (Scratch but with extra stuff) offers, you could probably make a program that fetches the digits of pi on demand from an external source (MIT has a collection of the first billion, and the record was 202.112 trillion at the time of creation), creating a virtually infinite pi program. So that's exactly what I did!

I downloaded a billion digits from MIT and split it into 500 pieces. I then hosted the digits on Netlify because MIT seems to have not implemented Cross-Origin Resource Sharing.

I then went crazy and coded the entire thing in a night. I might eventually Scratch-ify it and store the digits in variables if possible, but not today. I don't think TurboWarp links are allowed on Scratch, but if you want my Netlify digits, go to pi-digits.netlify.app/xaa for the first 2M, then xab for the next, then xac, eventually xba, all the way to xtg.

The program is compiled with TurboWarp Packager into an HTML file. If you want to see the final digits and the ending for yourself, you can download the HTML file and use the TurboWarp Unpackager to get the original code. I left the code easily hackable if you know what you're doing.

Venom: The Last Stand (v0.9.0)

This is a free fan game based on the Venom/Spider Man franchise. You will need to use arrow keys to move, and the keys 1, 2, and 3 to attack. The game was never completely finished/polished, and if it is broken, please double-click the green flag.

Are You a Good Driver? (v1.0.1)

This is a silly game remixed off of a Griffpatch project on Scratch that tests your driving skills.
